# Homebrew Tap for mc-notch

This is a Homebrew tap for [mc-notch](https://github.com/ImRLopezAG/mc-notch), a dynamic notch hub for macOS.

## Installation

```bash
brew tap ImRLopezAG/mc-notch
brew install --cask mc-notch
```

## What is mc-notch?

mc-notch transforms your Mac's notch into an interactive dynamic hub featuring:
- Music playback control (Apple Music, Spotify, browser media)
- File shelf with drag-and-drop
- Productivity tools (timers, clipboard, calendar)
- System-wide Now Playing detection
